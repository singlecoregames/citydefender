# City Defender — Game Design Document

> Missile Command × Active Incremental. 수동 요격으로 시작해, 점점 자동 포탑 군단을
> 지휘하는 사령관이 되어가는 4–5시간 분량의 인디 게임.
>
> 레퍼런스: **Missile Command**(코어 플레이), **Shelldiver**(진행 구조 — 능동형
> incremental, 분기형 스킬트리, 명확한 엔딩), **SNKRX**(아트 — 네온 도형 + 블룸).

---

## 1. 핵심 컨셉 (Core Pillars)

1. **Missile Command를 그대로**: 떨어지는 미사일을 클릭/탭 지점 폭발로 요격, 도시를 지킨다.
2. **수동 → 자동으로의 전이**: 초반엔 100% 수동 플레이. 중반부터 자동 포탑이 해금되며,
   후반엔 자동화가 주력이 되고 수동 사격은 강력한 보너스(+@)가 된다.
3. **능동형 incremental**: Shelldiver처럼 AFK가 아닌, 계속 플레이하며 4–5시간 안에
   엔딩을 보는 구조. 방치 요소는 최소화.
4. **읽히는 미니멀리즘**: SNKRX식 네온 도형. 모든 비주얼은 절차적으로 그려서
   외부 아트 에셋 없이 완성한다.

## 2. 게임 루프

```
┌─────────────────────────────────────────────────────┐
│  NIGHT (런, 2~5분)                                   │
│  웨이브 방어 → 적 격추로 Scrap 획득 → 밤 종료        │
│  (도시 생존 수에 따라 보너스, 전멸 시 조기 종료)      │
└───────────────┬─────────────────────────────────────┘
                ▼
┌─────────────────────────────────────────────────────┐
│  DAY (메타, 30초~2분)                                │
│  스킬트리에 Scrap/Cores/Data 투자                    │
│  도시 수리, 포탑 배치/교체, 다음 밤 준비             │
└───────────────┬─────────────────────────────────────┘
                ▼ 다음 NIGHT (더 강한 웨이브)
```

- **Night(밤) = 1 런.** 밤 `n`은 `4 + floor(n/3)` 웨이브로 구성 (초반 2분 → 후반 5분).
- 도시가 전부 파괴되면 밤이 조기 종료되고 해당 밤 수익의 60%만 획득 (소프트 실패 — 진행은 항상 전진).
- 같은 밤을 재도전하는 게 아니라 **밤 번호는 계속 전진**하되, 스킬트리로 "시작 밤 선택"을
  해금하면 낮은 밤으로 내려가 farm 가능 (Shelldiver의 expedition 커스터마이즈에 대응).
- **엔딩**: Night 50 "The Swarm" 보스 밤 클리어 시 엔딩 연출. 이후 자유 플레이(Endless) 개방.

## 3. 코어 게임플레이 (Missile Command 충실 재현 + 변형)

### 3.1 플레이어 측
- 화면 하단의 **지면 자체가 방어 대상** — 지면 밴드가 기본 3칸(업그레이드 Districts로 최대 6칸)
  으로 분할되고 각 칸이 HP를 가진다. 적이 지면에 닿으면 착탄점의 칸이 피해를 입는다.
  칸이 잘게 쪼개질수록 피해가 국소화되어 유리하다. 포대 슬롯(시작 1개: 중앙 수동 캐논).
- **스태틱 필드 (기본 공격)**: **커서(터치 지점)를 그냥 따라다니는 원형 오라** — 클릭도
  드래그도 필요 없다. 오라는 주기적으로 **펄스**해 반경 안의 모든 적을 감전(기본 1
  데미지)시키고 잠시 40% 슬로우(0.6초). 쿨다운은 **원 테두리를 따라 차오르는 링
  프로그레스바**로 표시되고, 가득 차면 밝게 점등 — 사정권에 적이 없으면 펄스를 아껴
  두었다가 처음 들어온 적을 즉시 감전시킨다. **기본치는 의도적으로 약하다**(반경 5.5,
  주기 1.2초 — 2.0초는 손맛이 늘어졌다는 피드백으로 단축. 펄스당 1은 유지: 0.6
  파리티안은 "1펄스 = HP1 적 처치" 브레이크포인트를 깨서 폐기. 런칭값 9/0.9초는
  필드 단독으로 밤을 캐리해버렸음): 트리의 필드
  사다리(§4.2 ①)를 찍어야 주력 무기로 성장한다(만렙 반경 ≈12.4, 주기 ≈0.69초).
  **정전기는 위상(phase) 실드를 관통**한다: 필드는 위상 적에 대한 수동 카운터
  (포탑은 여전히 Doppler Tracking 필요). 보스는 슬로우 면역(하강이 곧 전투 타이머),
  데미지는 유효. 필드 킬은 콤보를 먹이지도 끊지도 않음.
- **수동 캐논 (서브 공격 = 준어빌리티급 강타)**: 탭한 지점으로 요격 미사일 발사 →
  도달 지점에서 **대형 폭발**(기본 반경 10, 데미지 3 — 한 방이 클러스터 하나를 정리.
  런칭값 13은 수동 한 방이 지배적이라 축소; Blast Radius/Wide Blast 사다리로 성장).
  탄창이 빡빡한 **버스트 도구**(기본 2발, 4초당 1발 재생)라 매 발이 판단이다 —
  필드가 못 따라가는 물량/고HP/급한 위협에 쓴다. 콤보와 Overcharge는 캐논 폭발에만
  붙는다(정밀 플레이의 보상 축). **홀드 연사는 없다** — 연사가 있으면 캐논이 "제2의
  일반 공격"으로 읽혀 필드와 역할이 겹친다(플레이테스트 피드백으로 제거). 조작 문법:
  **커서 = 지속 공격(필드), 탭 = 강타(캐논), 버튼 = 어빌리티** 3층 구조.
- 지면 칸은 HP를 가짐 (기본 1히트 파괴 → 업그레이드로 HP/실드/자동수리). 잔여 HP는
  칸 내부 게이지로 표시. **보스는 호버링 없이 계속 하강**하며, 지면에 닿으면 어느 칸이든
  즉시 전 칸 파괴 = 밤 패배 (보스전은 착지 전에 잡아야 하는 시간전).

### 3.2 적 유닛 (도형 언어)
| 적 | 도형 | 등장 | 행동 | HP |
|---|---|---|---|---|
| Ballistic | 붉은 사각형 | Night 1 | 상단에서 직선 낙하 | 1 |
| Swarmer | 작은 주황 사각형 | Night 3 | 빠르고 약함, 무리 스폰(2→4마리) | 1 |
| Splitter | 보라 다이아몬드 | Night 5 | 사망 시 스워머 2기로 분열 | 2 |
| Regenerator | 초록 사각형 | Night 7 | 잠시 안 맞으면 HP 재생 | 5 |
| Phase | 파랑 사각형 | Night 11 | 주기적으로 무적/조준 불가(위상) | 2 |
| Carrier | 큰 진홍 사각형 | Night 19 | 느리고 단단, 하강하며 스워머 사출 | 10 |
| Armored | 강철색 사각형 | Night 36 | 피격당 고정 데미지 감산(잽 무기 카운터) | 6 |
| Cruise | 노랑 가로형 기체 | Night 42 | 측면 진입, 사인 곡선 횡단 후 급강하 | 4 |
| MIRV | 분홍 다이아몬드 | Night 51 | 중고도에서 탄도탄 3기로 분열(분열 전 격추 = 이득) | 3 |
| Healer | 민트색 사각형 | Night 66 | 느리게 하강하며 주변 적 주기 회복(우선표적) | 6 |
| **Boss** | 거대 복합 도형 | 10밤마다 | HP 덩어리 + 미니언 사출, 착지 = 즉시 패배 | 다량 |

신규 4종(Armored/Cruise/MIRV/Healer)은 월드 2~4 콘텐츠: 각 월드 진입 램프가
끝난 몇 밤 뒤에 낮은 가중치로 데뷔해 서서히 흔해진다(캐리어 N12 데뷔가 만든
연속 패배 스파이크의 교훈 — waves.ts `enemyPool` 주석 참조).

적 HP/속도/수량은 밤 번호에 따라 지수 스케일 — 단 HP 지수는 **N4부터 시작**
(`HP × 1.13^max(0, n-4)`, 수량 `× 1.05^n`). 수동 전용 초반(첫 포탑 이전)은 기본
HP로 유지하고, 도구가 생기는 시점부터 램프가 물리도록 밸런스 시뮬로 튜닝.

**공중 분열/사출 가속 램프**: 분열(스플리터 사망)·사출(캐리어/보스)로 공중에서
태어나는 적은 **본래 속도의 35%로 시작해 2.5초에 걸쳐 가속**한다. 분열이 방금
펄스/포탄을 소모한 지점에 풀스피드 자식을 쏟아내면 대응 창이 없다는 플레이테스트
피드백의 해법 — 의도된 속도는 그대로 도달하되, 반응할 시간을 준다.

### 3.3 수동 플레이의 후반 가치 (+@ 설계)
자동화가 깡DPS를 담당하더라도 수동이 계속 의미 있도록:
- **Overcharge Shot**: 수동 사격은 항상 "현재 전체 포탑 DPS의 X%"를 추가 데미지로 갖는
  버스트 — 자동화가 강해질수록 수동 한 방도 같이 강해짐.
- **Static Link**: 필드 펄스도 같은 공식으로 "전체 포탑 DPS의 X%"를 추가 데미지로
  얻음 — 후반에도 커서를 얹는 손이 포탑 군단과 함께 강해진다. 위상 적 관통이
  필드 고유의 역할로 남아 자동화 시대에도 커서를 움직일 이유가 유지됨.
- **Combo Meter**: 수동 요격 연속 성공 시 전역 데미지/수익 배율 증가 (놓치면 감소).
- **Active Abilities**(스킬트리 해금): EMP(전체 슬로우), Mega Bomb(거대 폭발), Time Dilation
  (3초 슬로모) — 전부 수동 발동, 쿨다운제.
- 보스의 약점 코어는 수동 사격으로만 노출되는 패턴 1개씩 포함.

### 3.4 보스 (5회)
- **N10 Carrier**: 대형 사각형, MIRV 연속 사출. 파괴 시 첫 Core 다량.
- **N20 Hydra**: 분열 반복하는 다이아몬드 군체.
- **N30 Bastion**: 실드 재생 — 수동 Overcharge로 실드 브레이크 필요.
- **N40 Eclipse**: 화면 절반을 가리는 장막 + 장막 뒤 발사.
- **N50 The Swarm**: 전 적 유형 총동원 + 3페이즈. 클리어 = 엔딩.

## 4. Incremental 시스템

### 4.1 화폐 3종 (Shelldiver의 다중 화폐 점진 해금 구조)
| 화폐 | 획득 | 용도 | 해금 |
|---|---|---|---|
| **Scrap** (⬡ 노랑) | 적 격추, 밤 완료 보너스, 생존 도시당 보너스 | 스킬트리 대부분의 노드 | 처음부터 |
| **Cores** (◆ 시안) | 보스, 밤 첫 클리어, 도전 과제성 마일스톤 | 대형 해금 노드(포탑 종류, 포대 슬롯, 도시 슬롯, 어빌리티) | Night 10 보스 |
| **Data** (▣ 마젠타) | "퍼펙트 웨이브"(피해 0), 콤보 유지, 수동 요격 비율 | 자동화 지능 트리(타게팅 AI, 우선순위, 효율) | Night 20 이후 |

Data를 "잘한 플레이"에서만 주는 것이 핵심 — 자동화 게임이 되어도 능동 플레이가 보상받는 축.

### 4.2 스킬트리 (3경로 + OPS 스텁, 약 62노드 — Nodebuster/Shelldiver 재편)
플레이테스트에서 옛 5방향 부채꼴이 **동시 선택지 20개+**를 쏟아내 재편했다.
중앙 COMMAND에서 **긴 트렁크 3개 + 짧은 곁가지**로 뻗는다:

```
                [BATTERY — 필드 트렁크 + 캐논/레이저 곁가지]
                      │
   [OPS ◆ 어빌리티] ─ [CORE] ─ [FOUNDATION — 경제 트렁크 + 도시/지원 곁가지]
                      │
                [WORKSHOP — 포탑 사다리 + 포탑별 곁가지 + 사격통제]
```

트리를 읽히게 하는 3가지 규칙 (테스트로 강제됨):
1. **소유 = 해금**: 선행 노드를 1레벨이라도 사면 자식이 열린다. 레벨 게이트
   ("선행을 Lv 2~3까지")는 플레이테스트에서 비직관적이라는 피드백으로 제거 —
   깊이 페이싱은 아래 가격 밴드가 전담한다 (다음 링은 항상 ≥1.5배 비싸서
   러시가 가능은 하되 최적은 아님).
2. **동일 효과 반복은 직렬로**: Turret Power → … → Turret Power II처럼
   같은 효과의 더 비싼 사본은 한 사다리 위의 다음 발판 — 나란히 놓고 고르는
   선택지가 아니다.
3. **가격 밴드 계단**: 링 d의 최소가가 링 d−1 최대가의 ≥1.5배 (겹침 금지).
   d1 20–80 / d2 250–300 / d3 490–700 / d4 1330–1960 / d5 3150–4550 /
   d6 7000–9800 / d7 15400–19600. 티어 2~4는 월드 수입 계단 가격 유지.
   딥 링(d5+)은 의도적으로 월드 1 수입을 초과 — 월드 2 진입기의 캐치업 콘텐츠.

**안개 공개**: 선행을 시작(1레벨)하지 않은 노드는 아예 숨김 — 트리는 소유의
한 발 앞까지만 펼쳐진다 (티어 노드는 실루엣 티저 유지, 단 실루엣으로 가는
연결선은 그리지 않는다 — 허공에 뜬 선처럼 읽힘). 어빌리티는 OPS 스텁의
◆ 토큰 해금이라 스크랩 프런티어와 경쟁하지 않는다. OPS 체인은 **Mega Bomb이
선두** — 첫 보스 토큰(N10)으로 공격형 어빌리티를 바로 살 수 있다
(플레이테스트: 워크숍 깊숙이 있던 시절 N27까지 공격 어빌리티 0개) → EMP →
Free Fire → 쿨다운 노드들.
**각 포탑 곁가지의 최외곽 캡스톤 = 트윈 배치 노드** (같은 포탑을 반대 측면에 한 기 더).
월드 진입 난이도: 적 HP의 월드 계단(×4/×16/×40)은 **진입 5밤에 걸쳐 기하 램프**로
들어온다 (월드 첫 밤 절벽 방지 — worldEntryRampNights).

**① Gunner (수동, 18노드)** — Scrap 중심. **필드 사다리가 트렁크, 캐논은 측면**
- 필드(기본 공격) 사다리: **Static Charge**(링1 게이트웨이 20⬡, 펄스 데미지 +0.3×5) →
  **Wide Field**(반경 +10%×5) → **Pulse Cycle**(펄스 쿨다운 −7%×5) →
  **Field Coils**(반경 +12%·쿨다운 −6%×3) → **Static Link**(펄스에 포탑 총 DPS의
  +4%×5 — §3.3 키스톤).
- 캐논(서브) 측면: Blast Radius/Wide Blast(폭발 반경), Warhead/Heavy Warhead(데미지),
  Overcharge Shot, Combo Memory. 탄약/장전 하드웨어는 워든 쪽(Drum Magazine →
  Autoloader → Fast Intercept)으로 이동 — 버스트 도구의 강화는 사이드 투자.

**② Automation (26노드)** — Cores로 해금, Scrap/Data로 강화
- 포대 슬롯 +1 ×4 (◆)

  → **Missile Pod**(유도탄) → **Railgun**(관통 저격)

### 4.3 비용/수익 곡선 (초기값 — 시뮬레이터로 튜닝)
- 노드 비용: `baseCost × costGrowth^레벨` (baseCost는 §4.2의 깊이 밴드를 따름).
  표시/청구 가격은 **친화 단위로 스냅**: 100 미만은 5단위, 그 이상은 10단위(뒷자리 0).
- 격추 보상: `tier보상 × 1.12^night × (1 + 콤보보너스)`.
- 밤 완료 보너스: `25 × 1.15^night × (생존 도시/전체 도시)`.
- 목표 페이싱 검증은 **헤드리스 밸런스 시뮬레이터**(§구현 플랜)로 자동화한다.

## 5. 아트 디렉션 (SNKRX 스타일 — 픽셀아트 + 은은한 글로우)

핵심은 **"매끄러운 벡터 네온"이 아니라 "깔끔한 픽셀아트 도형"**. SNKRX처럼
어두운 체크무늬 바닥 위에 또렷한 픽셀 도형이 놓이고, 글로우는 거들 뿐 주인공이 아니다.

- **픽셀화 파이프라인**: Three.js를 저해상도 버퍼(`RESOLUTION ≈ 360p`)에 렌더한 뒤
  CSS `image-rendering: pixelated`로 nearest-neighbor 업스케일 → 모든 도형이 굵은
  픽셀 그리드에 스냅. `antialias: false`. (이 한 줄 상수로 픽셀 크기 조절.)
- **배경**: 거의 같은 명도의 두 **무채색** 회색(`#262626`/`#1f1f1f`) **체크무늬 바닥**
  + 밝은 회색(`#404040`) 아레나 프레임(바닥선/양쪽 벽). 2×2 픽셀 텍스처를 nearest로
  타일링 (외부 에셋 0).
- **글로우는 은은하게**: `UnrealBloomPass` 강도 0.55 / threshold 0.55 — 밝은 코어만
  살짝 발광. 과한 네온 금지.
- **팔레트**: SNKRX 원작의 7색을 그대로 사용 — fg `#dadada` · yellow `#facf00` ·
  orange `#f07021` · red `#e91d39` · green `#8bbf40` · blue `#019bd6` ·
  purple `#8e559e`. UI(DOM)와 스킬트리도 같은 팔레트로 통일.
- **도형 언어**: SNKRX처럼 **도형이 진영을 말한다** — 플레이어측 탄환/요격탄 = 원,
  적 = 둥근 사각형(스플리터는 45° 다이아몬드, 보스는 천천히 회전), 구조물(도시/캐논/
  포탑/지원 건물) = 둥근 블록. 포탑과 건물은 **같은 칩 셰입**(6×7.5)을 공유하고
  색 + 룬으로만 구분한다. 색은 양 진영이 같은 팔레트를 공유.
- **종류 룬**: SNKRX의 룬 문자처럼, 포탑/지원 건물 몸통 가운데 **7×7 픽셀 글리프**를
  어두운 잉크로 새겨 종류를 구분한다 (개틀링 = 삼열 총열, 플랙 = 방사 X, 레이저 = 빔,
  미사일 = ↑, 레일건 = », 테슬라 = Z / 하베스터 = 보석, 실드 = 방패, 리페어 = +,
  레이다 = 조준환, 재머 = 파형, 디코이 = !). 캔버스 텍스처로 절차 생성 — 외부 에셋 0.
- **드롭 섀도**: SNKRX처럼 **모든 스프라이트가 우하단으로 하드 섀도**를 떨어뜨린다
  (월드 기준 고정 오프셋 ≈ 렌더 4px, 반투명 검정 — 아래 깔린 대상 위에도 자연스럽게
  드리움). DOM UI도 글로우 대신 같은 방향의 오프셋 섀도로 통일.
- **타격 피드백**: 피격 시 흰색 플래시(0.09s), 스폰 시 오버슈트 팝인, 사망 시 유닛
  색 파티클 팝, 트레일은 유닛 색의 어두운 톤.
- **HP/자원 게이지**: 몸통 자체가 게이지 — 테두리(월드 0.55 ≈ 1080p 기준 6px)는 몸통
  색으로 유지되고, 내부는 어두운 인셋 위로 남은 비율만큼 몸통 색이 아래에서부터
  채워진다(가득 = 통짜 도형). 내부는 **플랫 쿼드**(둥근 사각형 스케일 아님 — 대형
  저체력 몸통에서 뭉개져 보이던 문제 방지)이고 상한 캡으로 모서리가 림 밖으로 새지
  않는다. 적(HP)·도시(HP)·실드 제너레이터(남은 차지)에 공통 적용. 피격 플래시는
  채움까지 함께 번쩍이고, 페이즈 유령 적과 파괴된 도시 잔해는 게이지를 숨긴다.
- **겹침 정렬**: 적마다 고유 z 슬롯(2.0–2.4)을 부여해 겹칠 때 항상 한쪽이 명확히
  위로 그려진다. 아군 탄환/발사체는 2.5, 빔 2.7, 폭발 3.0으로 적 위 레이어.
- **이펙트**: 폭발 = SNKRX AoE 방식 — **밝은 흰색 additive 채움 원**이 착탄 순간
  최대 반경으로 나타나 잠시 유지된 뒤 **축소되며 소멸**(데미지 반경 로직과 동일:
  즉시 최대 → hold → shrink). 테두리는 **끊어진 호 4개(각 ~70°, 두껍게)**가 서로
  다른 속도·방향으로 빠르게 공전. 운석(적)은 낙하 중 각자 다른 속도로 회전(텀블링),
  미사일 = 짧은 픽셀 트레일, 도시 피격 = 화면 셰이크. 콤보 = 캐논 주변 링 게이지.
- **CRT 필터**: 게임 화면(캔버스) 위에만 얹히는 디스플레이 해상도 CSS 오버레이 —
  가로 스캔라인(3px 주기) + 아주 옅은 RGB 어퍼처 그릴 + 코너 비네트 + 9초 주기의
  희미한 롤링 스캔 밴드. DOM UI(HUD/트리)는 오버레이 위 레이어라 필터의 영향을 받지
  않는다. 순수 합성 CSS라 셰이더 패스 없이 모바일에서도 공짜. `prefers-reduced-motion`
  시 롤링 밴드 비활성.
- **사운드**: ZzFX(절차적 SFX)로 레트로 톤 통일. BGM은 후순위(앰비언트 1~2트랙).

## 6. UX / 플랫폼별 입력
- **데스크톱**: **커서 위치 = 스태틱 필드**(호버만으로 작동), 클릭 = 캐논 강타.
  1/2/3 = 어빌리티. Space = 일시정지. 휠 = (Endless) 배속.
- **모바일**: **터치 지점 = 필드**(드래그로 이동, 포인터 캡처로 손가락이 캔버스를
  벗어나도 유지; 손을 떼면 오라는 마지막 위치에 머무름), 탭 = 캐논 강타. 하단
  어빌리티 버튼. 도시/포대는 화면 최하단에서 약간 띄워 손가락 가림 방지.
- 필드 펄스 쿨다운은 오라 테두리의 링 프로그레스바로 표시 (가득 = 점등 = 발사 대기).
- 스킬트리/메뉴는 DOM 오버레이 (캔버스 UI보다 접근성/반응형/i18n에 유리).
- 세이브: 자동 (밤 종료마다 + 30초 주기). 슬롯 1개 + 클라우드(Steam).
