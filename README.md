# City Defender

A Missile Command–based active incremental game. Defend your cities by hand at first;
build an army of automated turrets as you climb a sprawling skill tree toward Night 50.

- **Engine**: Three.js (orthographic + bloom), TypeScript, Vite
- **Platforms**: Web, Steam (Electron + steamworks.js), iOS/Android (Capacitor)
- **Art**: SNKRX-style procedural neon shapes — zero external art assets
- **Scope**: ~4–5 hours to the ending, active (non-AFK) incremental
